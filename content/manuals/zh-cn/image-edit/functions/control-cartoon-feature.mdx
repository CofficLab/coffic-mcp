# 卡通形象控制

import { CodeExample } from '@coffic/cosy-ui';

## 功能描述

卡通形象控制利用AI的深度理解能力，分析参考卡通形象的特征、风格和个性，然后根据用户的创意描述生成新的图像。这种功能特别适合为卡通角色创建新的场景、表情、动作或服装，让卡通形象更加生动和有趣。

## 使用方法

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/control_cartoon_feature_1.png",
  prompt: "卡通形象小心翼翼地探出头，窥视着房间内一颗璀璨的蓝色宝石。",
  function: "control_cartoon_feature"
})
```

## 参数说明

| 参数 | 类型 | 必需 | 描述 |
|------|------|------|------|
| `imageUrl` | string | ✅ | 参考卡通形象图片URL |
| `prompt` | string | ✅ | 生成内容描述，说明要生成的新场景或内容 |
| `function` | string | ❌ | 固定为"control_cartoon_feature" |
| `n` | number | ❌ | 生成图片数量，默认1 |
| `model` | string | ❌ | 使用的模型，默认wanx2.1-imageedit |
| `dashScopeApiKey` | string | ❌ | DashScope API密钥 |
