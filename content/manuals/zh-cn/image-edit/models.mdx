# 模型查询

import { CodeExample } from '@coffic/cosy-ui';

`imageedit_models` 工具用于查询图像编辑功能支持的模型列表，帮助用户了解可用的模型和它们的能力。

## 🚀 使用方法

```typescript
// 查询所有支持的模型
imageedit_models()
```

## 📋 返回结果

工具会返回所有支持的图像编辑模型，包括：

- **模型名称** - 模型的唯一标识符
- **模型描述** - 模型的详细说明
- **支持能力** - 模型支持的所有编辑功能
- **推荐状态** - 是否为推荐使用的模型

## 🎯 当前支持的模型

### wanx2.1-imageedit

**描述**: 通用图像编辑模型，支持多种编辑任务

**推荐状态**: ⭐ 推荐使用

**支持的能力**:
- 整体风格化
- 局部风格化
- 指令编辑
- 局部重绘
- 去文字水印
- 图像修复
- 扩图
- 图像超分
- 图像上色
- 线稿生图
- 卡通形象控制

## 💡 使用建议

1. **优先使用推荐模型** - 推荐模型经过优化，效果更好
2. **根据功能选择** - 不同模型可能在不同功能上有优势
3. **定期查询更新** - 模型列表可能会更新，建议定期查询

## 🔧 技术特点

- 实时查询最新的模型信息
- 提供结构化的模型数据
- 支持模型能力对比
- 集成到MCP系统中，支持类型安全
