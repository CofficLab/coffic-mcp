# 图像编辑工具

图像编辑工具提供了强大的图像处理能力，支持11种不同的编辑功能，包括风格化、局部重绘、扩图、超分等。

## 可用工具

- **[图像编辑](./create.mdx)** - 主要的图像编辑功能
- **[模型查询](./models.mdx)** - 查询支持的编辑模型
- **[任务状态](./status.mdx)** - 查询编辑任务状态

## 核心功能

- **整体风格化** - 将整个图像转换为指定风格
- **局部风格化** - 对图像的特定区域应用风格
- **指令编辑** - 根据文字描述修改图像内容
- **局部重绘** - 使用蒙版指定区域进行重绘
- **去文字水印** - 智能去除图像中的文字和水印
- **图像修复** - 修复图像中的缺陷和损坏
- **图像扩图** - 智能扩展图像边界
- **图像超分** - 提升图像分辨率和质量
- **图像上色** - 为黑白图像添加颜色
- **线稿生图** - 将线稿转换为完整图像
- **卡通形象控制** - 控制卡通形象的生成

## 技术特点

- 基于阿里云DashScope API
- 支持异步任务处理
- 提供完整的错误处理
- 支持多种图像格式
- 可配置的生成参数
