# 文生图工具

文生图工具提供了强大的文本到图像生成能力，支持多种AI模型和丰富的参数配置，能够根据文字描述生成高质量的图像。

## 可用工具

- **[文生图](./create.mdx)** - 主要的文本转图像功能
- **[模型查询](./models.mdx)** - 查询支持的生成模型
- **[任务状态](./status.mdx)** - 查询生成任务状态

## 核心功能

- **文本描述生成** - 根据详细的文字描述生成图像
- **多种尺寸支持** - 支持从512x512到1440x1440的多种分辨率
- **批量生成** - 一次可以生成多张图像
- **模型选择** - 支持多种AI模型，各有特色
- **参数调优** - 可配置的生成参数，满足不同需求

## 技术特点

- 基于阿里云DashScope API
- 支持异步任务处理
- 提供完整的错误处理
- 支持多种图像格式
- 可配置的生成参数
- 随机种子控制

## 应用场景

- **创意设计** - 为设计项目生成概念图
- **内容创作** - 为文章、博客生成配图
- **原型设计** - 快速生成产品原型图
- **艺术创作** - 探索AI艺术的可能性
- **教育培训** - 为教学内容生成可视化素材

## 使用流程

1. **选择模型** - 使用 `text2image_models` 查询可用模型
2. **提交任务** - 使用 `text2image` 工具生成图像
3. **获得任务ID** - API返回任务标识
4. **等待执行** - 通常需要10-20秒
5. **查询状态** - 使用 `text2image_status` 工具
6. **获取结果** - 任务完成后获得生成的图像URL
