# 模型查询

import { CodeExample } from '@coffic/cosy-ui';

`text2image_models` 工具用于查询文生图功能支持的模型列表，帮助用户了解可用的模型和它们的特点。

## 🚀 使用方法

```typescript
// 查询所有支持的模型
text2image_models({})
```

## 📋 返回结果

工具会返回所有支持的文生图模型，包括：

- **模型名称** - 模型的唯一标识符
- **模型描述** - 模型的详细说明
- **推荐状态** - 是否为推荐使用的模型
- **适用场景** - 模型的最佳使用场景

## 💡 使用建议

1. **优先使用推荐模型** - 推荐模型经过优化，效果更好
2. **根据需求选择** - 快速生成选择极速版，高质量选择专业版
3. **考虑成本因素** - 极速版性价比更高，适合批量生成
4. **定期查询更新** - 模型列表可能会更新，建议定期查询

## 🔧 技术特点

- 实时查询最新的模型信息
- 提供结构化的模型数据
- 支持模型对比和选择
- 集成到MCP系统中，支持类型安全
