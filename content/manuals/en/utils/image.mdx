Image editing toolset based on DashScope API, supporting 11 different editing functions including stylization, image restoration, super-resolution, colorization, doodle-to-image, and more.

## üöÄ Supported Function Types

### 1. Global Stylization (`stylization_all`)
Convert the entire image to a specified style.

```typescript
imageedit({
  imageUrl: "https://example.com/image.jpg",
  prompt: "Convert to French picture book style",
  function: "stylization_all"
})
```

### 2. Local Stylization (`stylization_local`)
Convert specific parts of the image to a specified style without requiring masks.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/stylization_local_1.png",
  prompt: "Transform the house into wooden board style.",
  function: "stylization_local"
})
```

### 3. Description Editing (`description_edit`)
Modify specific content in the image based on text descriptions without requiring masks.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/description_edit_2.png",
  prompt: "Change the girl's hair to red.",
  function: "description_edit"
})
```

### 4. Masked Description Editing (`description_edit_with_mask`)
Perform precise redrawing based on text descriptions and mask-specified areas.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/description_edit_with_mask_3.jpeg",
  prompt: "A ceramic rabbit holding a ceramic flower.",
  function: "description_edit_with_mask",
  maskUrl: "http://wanx.alicdn.com/material/20250318/description_edit_with_mask_3_mask.png"
})
```

### 5. Watermark Removal (`remove_watermark`)
Remove watermarks and text from images.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/remove_watermark_1.png",
  prompt: "Remove text from the image",
  function: "remove_watermark"
})
```

### 6. Image Inpainting (`inpainting`)
Repair defects in images, requires mask provision.

```typescript
imageedit({
  imageUrl: "https://example.com/damaged_image.jpg",
  prompt: "Repair scratches and stains in the image",
  function: "inpainting",
  maskUrl: "https://example.com/damage_mask.jpg"
})
```

### 7. Image Expansion (`expand`)
Expand images in four directions with customizable scaling ratios.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/expand_2.jpg",
  prompt: "A green fairy",
  function: "expand",
  topScale: 1.5,
  bottomScale: 1.5,
  leftScale: 1.5,
  rightScale: 1.5
})
```

### 8. Super Resolution (`super_resolution`)
Improve image resolution with customizable upscaling factors.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/super_resolution_1.jpeg",
  prompt: "Image super resolution.",
  function: "super_resolution",
  upscaleFactor: 2
})
```

### 9. Image Colorization (`colorization`)
Add colors to black and white images based on prompt descriptions.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/colorization_1.jpeg",
  prompt: "Blue background, yellow leaves.",
  function: "colorization"
})
```

### 10. Doodle to Image (`doodle`)
Convert sketches or doodles into complete images.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/doodle_1.png",
  prompt: "Nordic minimalist style living room.",
  function: "doodle"
})
```

### 11. Cartoon Feature Control (`control_cartoon_feature`)
Generate images by referencing cartoon characters.

```typescript
imageedit({
  imageUrl: "http://wanx.alicdn.com/material/20250318/control_cartoon_feature_1.png",
  prompt: "A cartoon character carefully peeks out, gazing at a brilliant blue gem inside the room.",
  function: "control_cartoon_feature"
})
```

## üîç Mask Requirements

- **Requires Mask**: `description_edit_with_mask`, `inpainting`
- **Optional Mask**: `stylization_partial` (in some cases)
- **No Mask Required**: `stylization_all`, `stylization_local`, `description_edit`, `remove_watermark`, `expand`, `super_resolution`, `colorization`, `doodle`, `control_cartoon_feature`

## üìñ Usage Workflow

1. **Submit Editing Task**: Use the `imageedit` tool
2. **Get Task ID**: API returns task identifier
3. **Wait for Execution**: Usually takes about 15 seconds
4. **Query Status**: Use the `imageedit_status` tool
5. **Get Results**: Obtain generated image URLs after task completion

## üõ†Ô∏è Available Tools

- **`imageedit`** - Main image editing tool
- **`imageedit_models`** - Query supported models
- **`imageedit_status`** - Query task status

## ‚ö†Ô∏è Important Notes

- Image URLs must be accessible public links
- Mask images are used to specify editing areas, required for certain function types
- All tasks are executed asynchronously, requiring waiting and status queries
- Supports batch generation of multiple images (via `n` parameter)
- Expand function supports custom expansion ratios in four directions
- Super-resolution function supports custom upscaling factors to improve image resolution
- Colorization function is specifically designed for adding colors to black and white images
- Watermark removal function is specifically designed for removing watermarks and text content
- Masked description editing provides the most precise area control

## üîß Technical Implementation

All functions are implemented based on DashScope API, supporting asynchronous task processing to ensure long-running editing tasks can complete normally. The toolset is fully integrated into the MCP system, supporting type-safe parameter validation and error handling.
