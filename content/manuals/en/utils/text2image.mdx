Text-to-image tools based on DashScope API, supporting multiple models and parameter configurations, capable of generating high-quality images from text descriptions.

## üöÄ Main Features

### Text to Image (`text2image`)
Generate images based on provided text descriptions, supporting custom dimensions, quantities, and model selection.

```typescript
text2image({
  prompt: "A cute little cat sitting in a garden, bright sunshine, flowers blooming",
  size: "1024*1024",
  n: 1,
  model: "wan2.2-t2i-plus"
})
```

### Model Query (`text2image_models`)
Query supported text-to-image models to understand the characteristics and recommendations of each model.

```typescript
text2image_models({})
```

### Task Status Query (`text2image_status`)
Query the execution status of text-to-image tasks to obtain generated image results.

```typescript
text2image_status({
  task_id: "your_task_id_here"
})
```

## üìñ Usage Workflow

1. **Submit Generation Task**: Use the `text2image` tool
2. **Get Task ID**: API returns task identifier
3. **Wait for Execution**: Usually takes about 15 seconds
4. **Query Status**: Use the `text2image_status` tool
5. **Get Results**: Obtain generated image URLs after task completion

## üîß Technical Features

### Image Dimension Support
- **Pixel Range**: [512, 1440]
- **Maximum Resolution**: 2 million pixels
- **Common Dimensions**: 1024*1024, 1024*768, 768*1024
- **Custom Combinations**: Supports arbitrary width-height combinations

### Random Seed Control
- **Seed Range**: [0, 2147483647]
- **Stability**: Using the same seed can generate relatively stable content
- **Batch Generation**: Supports generating incremental seeds for multiple images

### Asynchronous Task Processing
- **Non-blocking**: Supports long-running generation tasks
- **Status Queries**: Real-time query of task execution progress
- **Error Handling**: Comprehensive error handling and status feedback
